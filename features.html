Quick bar

The QuickBar contains known leading indices and frequently traded securities. Click on an icon and navigate through the menus to open a chart or other documents.

- Click on All symbols to open a price page

- Click on the name of an icon to open a chart

- Click with the right mouse button on the name of a symbol to select the desired document type

Symbol Search

To find other securities, use Symbol Search from the toolbar or with File > Symbol Search.

- Select check boxes in the dialog before searching to limit your results

- Click on category names after searching to filter your results

- Select one or more symbols and click Open to choose a document type

Virtual Pages

With our virtual pages, you can add a set of pre-defined market overviews to your workspace.

- Click File > Open Virtual Pageto display the dialog Virtual Pages

- Click on the desired virtual page, for example World Overview World Overview

- The virtual page will be added to your workspace behind the active virtual page and is immediately shown on the screen

Compresion

Change the data compression of a chart

- Select the drawing tool on the toolbar (for example: Fibonacci Fan)

- Click on the chart for the first reference point and hold the mouse button down

- Move the mouse pointer to the second reference point and release the mouse button
